- WordPress 6.6 or higher

- Active internet connection

## Installation

### From WordPress Admin

1. Go to `Plugins > Add New`
2. Search for "Recompal"
3. Click "Install Now"
4. Activate the plugin
5. Plugin will auto-register your store

### Manual Installation

1. Download the plugin zip file
2. Upload to `/wp-content/plugins/` directory
3. Extract the zip file
4. Activate through WordPress admin
5. Plugin will auto-register your store

## Configuration

After activation, configure your plugin:

1. Navigate to `Recompal` in WordPress admin
2. Customize appearance settings
3. Configure data sources
4. Set up billing if needed
