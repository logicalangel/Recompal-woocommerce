Your AI Sales Assistant for WooCommerce Stores - Boost sales with intelligent product recommendations and automated customer support.

== Description ==

Recompal is an AI-powered sales assistant that helps WooCommerce store owners increase conversions and provide better customer experiences. 

**Key Features:**

* **AI-Powered Chat Widget** - Engage customers with intelligent conversations
* **Smart Product Recommendations** - Suggest relevant products based on customer behavior
* **Automated Customer Support** - Answer common questions instantly
* **Conversation Analytics** - Track customer interactions and gain insights
* **Easy Integration** - Works seamlessly with your WooCommerce store

**How It Works:**

1. Install and activate the plugin
2. Your store will be automatically registered with Recompal
3. Configure your AI assistant's appearance and behavior
4. Start engaging with customers through the chat widget

**Requirements:**

* Active WooCommerce installation
* Internet connection for API communication
* WordPress 6.6 or higher
* PHP 7.4 or higher

== Installation ==

**Automatic Installation:**

1. Log in to your WordPress admin panel
2. Navigate to Plugins > Add New
3. Search for "Recompal"
4. Click "Install Now" and then "Activate"
5. Your store will be automatically registered

**Manual Installation:**

1. Download the plugin zip file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the downloaded zip file and click "Install Now"
5. Activate the plugin
6. Your store will be automatically registered

**Post-Installation:**

1. Navigate to Recompal in the WordPress admin menu
2. Configure your AI assistant settings
3. Customize the appearance to match your brand
4. Add your product knowledge base

== Frequently Asked Questions ==

= Does this work with my theme? =

Yes, Recompal works with any WordPress theme that supports WooCommerce.

= Is WooCommerce required? =

Yes, WooCommerce must be installed and active for Recompal to function.

= How does the AI understand my products? =

Recompal automatically syncs your WooCommerce product catalog, including names, descriptions, categories, prices, and images.

= Can I customize the chat widget appearance? =

Yes, you can customize colors, position, and behavior from the Appearance settings page.

= Is my data secure? =

Yes, all communication is encrypted via HTTPS. We follow WordPress and WooCommerce security best practices.

= Does this slow down my website? =

No, the chat widget loads asynchronously and product syncs happen in the background without affecting your site performance.

= What happens to my data if I uninstall? =

All plugin options and API keys are automatically removed from your database when you uninstall the plugin.
